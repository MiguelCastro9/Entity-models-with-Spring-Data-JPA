# Overview

Entity modeling with all Spring Data JPA mappings, <code>@OneToOne</code>, <code>@ManyToOne</code>, <code>@OneToMany</code> and <code>@ManyToMany</code>

## Execution

Tools:
<li> Java 17 </li>
<li> SpringBoot 3.2.0 </li>
<li> Spring Data JPA </li>
<li> Docker </li>
<li> MySQL </li>
